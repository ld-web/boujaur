在這個小網站會有一些介紹在台灣學中文時期的文章。

這個專案用 NextJS 框架做的，因為代表一個在做網站的領域特別有趣的工具。

而且，以後會提供這些文章的中文版本，這個專案的結構目前應該已經準備好了收到。

## NextJS 跟 MDX 處理

文章都用 MDX 的語言，因為會讓在一個內容裡面寫 JSX 的標籤很容易。

因次，這個專案用 next-mdx-remote 軟體包。
